# OpenPerceptionLab

A multi-stage perception and SLAM learning project.

## Features

- Object Detection
- Semantic Segmentation
- Monocular Depth Estimation
- Visual SLAM
- LiDAR SLAM
- Sensor Fusion
- 3D Reconstruction
- Edge Deployment

## Roadmap

Level1: Visual Perception  
Level2: Depth Estimation  
Level3: Multi-view Geometry  
Level4: Visual SLAM  
Level5: LiDAR SLAM  
Level6: Sensor Fusion  
Level7: 3D Reconstruction  
Level8: Deployment
